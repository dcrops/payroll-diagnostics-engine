Chase Risk & Compliance (CRC)
System for analysing payroll and operational data to surface hidden risks

Problem

Payroll systems process transactions but don’t surface hidden inconsistencies or risks that accumulate over time.

Solution

A modular diagnostics system that ingests payroll data, applies rule-based analysis, and generates structured outputs highlighting potential issues.

Architecture
Data ingestion layer
Validation layer
Rule engine (multi-domain)
Findings generation
Reporting (Markdown → PDF)
Key Features
Rule-based anomaly detection
Structured outputs with evidence
Modular design for extensibility
Reporting layer for business users
AI Direction (important but honest)

A prototype RAG-based copilot has been developed to support system exploration, with future plans to integrate AI-assisted explanations.

Tech Stack

Python, Pandas, FastAPI (if used), Markdown reporting, etc.
